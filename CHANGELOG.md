# Changelog

## Unreleased

No unreleased changes recorded after `0.6.0`.

## 0.6.0 - 2026-04-06

- Expanded the running MCP registry from the `v0.5.0` install/file/system baseline to a 14-tool preview surface.
- Added entity-browsing tools: `list-entity-systems`, `find-entity`, `describe-entity`, and `list-entity-links` over the curated core inspection seam.
- Added grouped helper reports: `report-diplomacy-war-flow`, `report-diplomacy-graph`, and `report-religion-links` over representative install files and stable grouped `eu5miner` domain APIs.
- Added conservative mod workflow tools: `plan-mod-update` for dry-run planning and `apply-mod-update` for explicit confirmed writes.
- Added a real stdio MCP transport plus the `describe-server` self-description tool over shared runtime metadata and the live registered descriptor set.
- Hardened the runtime contract with stricter input schemas, explicit write confirmation, unified version and transport metadata, and registry validation for duplicate or missing tool declarations.
- Aligned package and release docs to the coordinated `eu5miner` `v0.6.0` core dependency tag.

## 0.5.0 - 2026-04-05

- First public preview release.
- Added a typed local MCP shell and CLI over stable `eu5miner` install, file, and system-report seams.
- The initial preview tool surface exposed `inspect-install`, `list-files`, `list-systems`, and `report-system`.
- Public-facing repo metadata now reflects the preview posture, MIT licensing, and a minimal responsible-disclosure policy.
- The downstream core dependency was pinned to the coordinated `eu5miner` `v0.5.0` release tag.
