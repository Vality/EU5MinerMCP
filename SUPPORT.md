# Support

EU5MinerMCP is a small public-preview project, so support is best effort.

## Getting Help

- Open a GitHub issue for reproducible bugs or concrete feature requests.
- Include your `eu5miner-mcp` version, Python version, OS, and exact command or input.
- Keep reports install-independent when possible, or clearly state when a local EU5 install is required.

## Security

Do not report security issues in public issues.

See `SECURITY.md` for the current vulnerability reporting process.
