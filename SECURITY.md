# Security Policy

## Supported Versions

The current supported line is `0.6.x`.

This repository is still a small public-preview project, so security support is best effort and focused on the latest preview state on `main`.

## Reporting A Vulnerability

Please do not open a public GitHub issue for suspected security problems.

- Prefer GitHub private vulnerability reporting for this repository when that option is available.
- If private reporting is not available, contact the maintainer privately through GitHub instead of posting details publicly.
- Include the affected version or commit, reproduction steps, expected impact, and whether the issue depends on a local EU5 install or mod content.

## Response Expectations

- Initial acknowledgement target: within 7 days.
- Fixes may land on `main` before a tagged release is cut.
- Public disclosure should wait until a fix or mitigation is available.
